Image search engine

User Story:

This is a image search engine powered by Google custom search API and mongodb. Search string return client image URLs, alt text and page urls for a set of relating images. User can paginate through the responses by adding a ?offset=2 parameter to the URL. User can also get a list of the most recently submitted search strings.

How to use?

  • Search for images example: https://example.com/imagesearch/lolcats%20funny?offset=10
  • Browse recent search queries : https://example.com/latest/imagesearch/
